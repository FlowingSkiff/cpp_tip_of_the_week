* **Did you know about C++26 proposal - `inplace_vector`?**

  * https://wg21.link/P0843

</p></details><details open><summary>Example</summary><p>

```cpp
int main() {
    std::inplace_vector<int, 2> v{};
    assert(v.empty());

    v.push_back(1);
    assert(1 == v.size());

    v.push_back(2);
    assert(2 == v.size());

    v.push_back(3); // throws
}
```
